# Nekosticker插件

* 该版本可以修改到自己有的皮肤上[也支持WS换肤],市面上需要另外ws插件依赖直接更换贴纸会导致自己皮肤显示失败.

* mysql support

* 全部可视菜单 + 国内物品API支持同步更新
