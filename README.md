# Nekosticker插件
* 移除多余拓展依赖,只需要PTaH拓展,支持Windows/Linux。
* 该版本可以修改到自己有的皮肤上[也支持WS换肤],公开版本插件只能使用另外的修改皮肤插件,直接更换贴纸会导致自己皮肤显示失败。
* mysql support
* 全部可视菜单 + 国内物品API支持同步更新。
